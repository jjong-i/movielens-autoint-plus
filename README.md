🎬 MovieLens AutoInt+ 추천 시스템 프로젝트
✅ 프로젝트 제출 항목
-**학습 노트북** (notebook.ipynb): 데이터 전처리부터 모델 학습까지의 전 과정 포함.

-**모델 정의 및 스트림릿** (.py): autoint_plus.py (고도화 모델) 및 app_final.py (시각화).

-**모델 가중치 (.h5)**: 기본 모델 및 고도화 모델의 학습 완료된 가중치.

**💡 주요 실험 내용 (성능 향상 시도)**

**-AutoInt+ 구현:** 기본 Self-Attention 구조에 MLP(Deep) 경로를 병렬로 추가하여 복잡한 피처 상호작용을 학습하도록 개선함.

-**프레임워크 전환**: PyTorch 기반의 베이스라인 코드를 TensorFlow/Keras로 변환하여 프로젝트 요건 충족.

-**최적화**: 에포크 10회 및 드롭아웃 적용을 통해 안정적인 학습 곡선 도출.
